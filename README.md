# 🍽️ Restaurant Website

A modern and fully responsive restaurant website built using HTML, CSS, Bootstrap, and JavaScript. This project was developed as part of my frontend development internship and showcases a complete multi-page restaurant website with responsive design and interactive features.

## 🚀 Live Demo

https://ismailhussainx0.github.io/Restaurant-Website/

## 📌 Features

* Responsive design for mobile, tablet, and desktop devices
* Modern and clean user interface
* Multi-page website structure
* Interactive navigation bar
* Menu page with food search functionality
* About Us page with restaurant information
* Contact page with form validation
* Google Maps integration
* GitHub Pages deployment

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (DOM Manipulation)
* Git & GitHub
